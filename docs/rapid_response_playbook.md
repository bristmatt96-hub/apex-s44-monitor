# Rapid Response Playbook

## Purpose

When news breaks, you have MINUTES to assess implications before the market moves. This playbook provides instant analysis frameworks.

---

## HEADLINE → IMPLICATIONS (60-Second Analysis)

### Restructuring/Distress Headlines

**"[Company] hires [Houlihan/Evercore/PJT/Lazard]"**
```
IMMEDIATE IMPLICATIONS:
- LME is CERTAIN within 3-6 months
- Bonds should trade to 70-85 range (if not already)
- CDS should widen 200-400bps
- Recovery analysis now relevant
- Check: Are you in ad hoc group? Should you be?

PRICE TARGET: 70-80 if currently higher
TIMELINE: LME announcement within 60-90 days
CDS: Restructuring event likely in 6-12 months
```

**"[Company] evaluating strategic alternatives / liquidity options"**
```
IMMEDIATE IMPLICATIONS:
- Formal distress acknowledgment
- Advisor already engaged (just not announced)
- Bonds should drop 5-15pts from current
- Sale/restructuring/new capital all possible
- Check: What's current price vs recovery?

PRICE MOVE: -5 to -15pts
TIMELINE: Formal announcement within 30-60 days
```

**"[Company] announces exchange offer"**
```
IMMEDIATE IMPLICATIONS:
- Terms matter enormously - READ THEM
- Early bird premium? How much?
- Participation threshold?
- Exit consent stripping covenants?
- What happens to holdouts?

KEY QUESTIONS (answer in 5 min):
1. What's the exchange ratio?
2. Is participation mandatory (Scheme) or voluntary?
3. What do holdouts get?
4. Does this trigger CDS Restructuring?
5. What's new paper trading at?
```

**"Ad hoc group forms at [Company]"**
```
IMMEDIATE IMPLICATIONS:
- Multiple creditors coordinating
- Usually means >30% of debt
- They have better information
- LME negotiation starting
- Non-members at disadvantage

ACTION: Immediately assess if you should join
CHECK: Who is advising the group? What are terms?
```

### Rating Headlines

**"[Company] downgraded to CCC"**
```
IMMEDIATE IMPLICATIONS:
- Forces CLO/insurance selling
- Technical pressure for 2-4 weeks
- May be buying opportunity if fundamentals not worse
- Fallen angel dynamics

PRICE MOVE: -3 to -8pts (technical)
OPPORTUNITY: Buy the technical if fundamental view positive
RISK: Further deterioration if fundamental
```

**"[Company] outlook revised to Negative"**
```
IMMEDIATE IMPLICATIONS:
- Downgrade likely in 6-12 months
- Usually signals concern not crisis
- Price impact: -1 to -3pts
- Sets up potential forced selling later

PRICE MOVE: -1 to -3pts
WATCH: Next rating action timeline
```

**"[Company] upgraded / outlook to Positive"**
```
IMMEDIATE IMPLICATIONS:
- Positive trajectory confirmed
- Check if already priced
- Potential CLO/index inclusion buying
- Usually +2 to +5pts

OPPORTUNITY: If not priced, buy for technical bid
RISK: May be priced in already
```

### Corporate Action Headlines

**"[Company] sells [asset] for $[X]"**
```
IMMEDIATE ANALYSIS (2 min):
1. Price vs book value → good/bad deal?
2. Use of proceeds → delever or dividend?
3. What's left → better or worse business?
4. Impact on EBITDA/leverage

GOOD: Proceeds to debt, reduces leverage
BAD: Proceeds to dividend, sells crown jewel
UGLY: Fire sale price, losing key asset
```

**"[Sponsor] takes $[X] dividend from [Company]"**
```
IMMEDIATE IMPLICATIONS:
- Value extraction from creditors
- Shows sponsor prioritizing returns over delever
- Bonds should widen 25-75bps
- Check covenant capacity → more coming?

SENTIMENT: Very negative
PRICE MOVE: -1 to -3pts
RED FLAG: For future LME behavior
```

**"[Company] announces acquisition of [target]"**
```
IMMEDIATE ANALYSIS:
1. Cash or debt funded?
2. Leverage impact?
3. Strategic sense?
4. Rating agency view?

DEBT-FUNDED: Negative for credit, check leverage
EQUITY-FUNDED: Neutral to positive
SYNERGY-DEPENDENT: Skeptical until proven
```

### Operational Headlines

**"[Company] reports Q[X] earnings"**
```
RAPID CHECKLIST (90 seconds):
1. EBITDA vs consensus → beat/miss?
2. Margin trend → expanding/compressing?
3. Guidance → raised/cut/maintained?
4. Leverage → up/down from prior Q?
5. Liquidity → any concerns?
6. Language → confident/cautious?

BEAT: Bonds +1-2pts, CDS tighter
MISS: Bonds -2-5pts, CDS wider
GUIDE DOWN: Bonds -3-7pts
```

**"[Company] issues profit warning"**
```
IMMEDIATE IMPLICATIONS:
- Earnings miss coming
- Guidance cut
- Bonds down 3-8pts typical
- Check: Temporary or structural?

KEY QUESTION: One-time or trend?
TEMPORARY: May be buying opportunity
STRUCTURAL: Sell, trajectory negative
```

**"[Company] CEO/CFO departs"**
```
IMMEDIATE IMPLICATIONS:
- Check circumstances → pushed or pulled?
- Restructuring CFO = LME coming
- Sudden departure = problem
- Orderly succession = neutral

RESTRUCTURING HIRE: Very negative
SUDDEN DEPARTURE: Negative, investigate
PLANNED TRANSITION: Neutral
```

## 5-MINUTE DECISION FRAMEWORK

When headline hits:

**MINUTE 1: Classify**
- Is this positive/negative/neutral?
- Is this priced or new information?
- Does this change trajectory?

**MINUTE 2: Quantify**
- How many points should this move price?
- What's current price vs fair value now?
- What's the trade?

**MINUTE 3: Check**
- What's the current bid/ask?
- Can I execute?
- Size available?

**MINUTE 4: Verify**
- Any conflicting information?
- Is the source reliable?
- Am I missing something?

**MINUTE 5: Decide**
- Trade now or wait for more info?
- Size: full position or starter?
- Set alerts for follow-up
